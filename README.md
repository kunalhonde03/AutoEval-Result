# AutoEval â€“ Smart Result Classification and Attendance Analysis System

AutoEval is a web-based system that simplifies the processing of online assessment results.
It converts raw Excel response data into structured student records, attendance information, and performance insights through an easy-to-use dashboard.

## Features

* Upload and process Excel result files
* Automatic classification by branch, division, and academic year
* Detection of absent students using roll-number comparison
* Real-time filtering, sorting, and performance statistics
* Export of filtered or branch-wise Excel reports
* Secure login-based access for teachers

## Tech Stack

HTML, CSS, JavaScript, XLSX library, Firebase.

## Purpose

To reduce manual effort, minimize errors, and speed up academic result analysis for educators.
